#### 什么是 Vibe Coding？

Vibe Coding 是 AI 时代的新型开发方式：

- 👨‍💻 你躺在人体工学椅上，眼睛盯着屏幕
- 🗣️ 用手机语音输入，告诉 AI 你想要什么
- 🤖 AI（如 Claude Code）自动生成代码、执行命令
- 📱 不用碰键盘，一切尽在掌控

这不是偷懒，这是**高效**。

---

#### 太白说：Vibe Coding 的标准装备

**太白说**是一款局域网语音输入同步工具，让你的手机变成电脑的第二输入设备。

**核心功能：**

| 功能 | 说明 |
|------|------|
| 📱 语音转文字 | 借助手机输入法的语音识别，说话即上屏 |
| 🖱️ 全屏触控板 | 手机变身触控板，躺着控制鼠标 |
| 📷 屏幕截图 | 一键截取电脑屏幕，手机查看支持双指缩放 |
| ⚡ 快捷操作 | Tab 切换、继续、回滚、编译、提交一键搞定 |
| ⚙️ 自定义设定 | 命令列表、常用语、自动发送全部可配置 |
| 🔧 正则替换 | 语音输入 "男主" 自动替换成 "张无忌" |

**特别适配 Claude Code 工作流：**

快捷按钮直接发送 "继续"、"回滚"、"/clear" 等指令，Tab 键切换编辑器标签页，ESC 键中断操作。躺着指挥 AI 干活，效率翻倍。

[开源] 太白说 - Vibe Coding 必备工具，手机语音输入同步到电脑

### 正文

#### 项目背景

作为一个长期使用 AI 辅助编程的开发者，我发现了一个问题：

- AI（如 Claude Code）可以自动写代码
- 但我还是要坐在电脑前打字发指令
- 长时间打字导致手腕酸痛、颈椎不适

于是我开始尝试 **Vibe Coding**：躺在椅子上，用手机语音输入指令。

但问题来了：手机的语音输入怎么同步到电脑？

**太白说**就是为此而生。

#### 功能特性

- **语音转文字**：借助手机输入法（推荐豆包）的语音识别，说话即上屏
- **全屏触控板**：手机变身触控板，单指滑动/单击/双击/双指右键
- **屏幕截图**：一键截取电脑屏幕，手机端支持双指缩放查看
- **快捷操作**：Tab 切换、继续、ESC、回滚、新任务、编译、提交
- **正则替换**：自定义替换规则，如 "男主" → "张无忌"
- **设定中心**：Logo 点击进入，可编辑命令列表和常用语

#### 技术栈

- Python + Flask
- pyautogui（键鼠模拟）
- PyInstaller（打包）

#### 使用方式

```bash
# 克隆仓库
git clone https://github.com/jinny76/taibai-talk.git

# 安装依赖
pip install -r requirement.txt

# 启动服务
python main.py -p 8888 --password yourpassword

# 手机扫描二维码即可使用
```

#### GitHub

https://github.com/jinny76/taibai-talk

欢迎 Star ⭐️ 和 PR！
